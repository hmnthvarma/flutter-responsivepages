# ChefKart Assignment

This app fetches menu data from a remote API, implements a dynamic cart management system, and displays detailed dish information including ingredients and preparation time.

## Features for now

    * Fetch ingredients data from API
    * Add items to cart with a single tap.
    * Increment/Decrement quantity controls replace the "Add" button dynamically.
    * Global cart summary bar appears conditionally when items are selected.
    
* **Detailed Views:** Separate detail screen fetching specific dish data (ingredients, appliances, time).
* **Smart Tagging:** Visual indicators for Veg/Non-Veg based on item attributes.
